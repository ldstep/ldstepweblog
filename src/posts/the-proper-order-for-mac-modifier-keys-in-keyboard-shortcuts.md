If you write about Mac keyboard shortcuts, there's a proper order for listing the modifier keys in a shortcut. The proper order is similar to how you see them at the bottom left of your keyboard.

Control (⌃), Option (⌥), and Command (⌘) always go in that order. The oddball is the Shift key (⇧), which is just before Command.

Keyboard Maestro follows this standard and presents its hot key field the same way.

![Keyboard Maestro hot key field](https://cdn.statically.io/gh/ldstep/my-blog-images/main/IMG_2604.jpg?width=800&format=webp)

Apple documents this order on their [Mac keyboard shortcuts support page](https://support.apple.com/en-us/102650), if you want the official word.
