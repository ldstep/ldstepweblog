June 10, 2026

Claude is my go-to AI tool. With the Drafts MCP connected, I can drop results directly into my Drafts inbox without copy and paste.

One thing I do regularly: have Claude summarize a long conversation, then send it straight to Drafts. From there, a Drafts action moves it to my Notes folder. It's especially useful when I've been vibe coding a script; the summary becomes a handy reference for how we got there.

Another thing I've started doing is asking Claude to proofread a draft directly from Drafts. No exporting, no copying text into a chat window — just ask, and Claude pulls the draft, reviews it, and sends back suggestions. It's a natural fit.

If you use both Claude and Drafts, it's worth setting up.
