I've shared my menubar apps before, but it's been a while so I figured it is worth a revisit. These are the ones I use every day and I keep them organized with [Thaw](https://github.com/stonerl/Thaw).

- **[Hazel](https://www.noodlesoft.com/)**: Automates file organization using rules I've set up. Folders stay clean without me touching them.
- **[Dropzone](https://aptonic.com/)**: Makes it easy to move files, upload to cloud services, or trigger repetitive tasks with a drag.
- **[PopClip](https://www.popclip.app/)**: Pops up a small menu whenever I select text -- copy, search, translate, whatever I've configured.
- **[ScreenFloat](https://www.screenfloatapp.com/)**: Lets me take a screenshot and keep it floating above other windows as a reference.
- **[Bitwarden](https://bitwarden.com/)**: My password manager. Open source, cross-platform, and I trust it.
- **[Tot](https://tot.rocks/)**: Quick scratch-pad notes that don't need to live anywhere.
- **[Maestral](https://maestral.app/)**: My Dropbox client. Also open source.
- **[Moom](https://manytricks.com/moom/)**: Handles all my window management.
