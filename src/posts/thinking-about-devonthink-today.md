June 14, 2026

This morning, I read an MPU forum [post](https://talk.macpowerusers.com/t/claude-meets-devonthink-an-assistant-for-my-research-library/45820) about [Devonthink](https://www.devontechnologies.com/apps/devonthink). I've had Devonthink starting with version 2 and currently have version 3 installed on my MacBook Pro. I don't often think about Devonthink because I never use it.

Many folks use it as a document warehouse for its search features, which is how I originally used it. I indexed all my most important folders so that I could easily search them. So Devonthink was basically just a layer on top of Finder. Then several years ago, I discovered HoudahSpot.

[HoudahSpot](https://www.houdah.com/houdahSpot/) is a power-user front end for Finder's underlying search engine. It lets me quickly find files by name, text content, metadata, date, and more. I have templates set up to search all the major folders that I use for all my most important files.

I no longer use Devonthink. It's sitting on my MacBook taking up space. I should probably just delete it.
