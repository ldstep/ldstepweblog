## Colophon

This blog is a static site built with [11ty](https://www.11ty.dev/), using [Nunjucks](https://mozilla.github.io/nunjucks/) templating engine, [HTML and CSS](https://htmlforpeople.com/), and deployed on [Vercel](https://vercel.com/). It automatically deploys whenever I commit changes to the [public GitHub repo](https://github.com/ldstep/ldstepweblog).

The domain is registered through [Hover](https://hover.com/).

When adding new posts, changing the design, or adding new features, that is done in [Zed](https://zed.dev).

New posts are written in Markdown in [MarkEdit](https://github.com/MarkEdit-app/MarkEdit) and added to Zed.
