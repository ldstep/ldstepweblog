Colophon

This blog is a static site built with 11ty, using HTML and CSS, and deployed on Netlify. It automatically deploys whenever I commit changes to the public GitHub repo. The domain is registered through Hover.

When adding new posts, changing the design, or adding new features, that is done in VS Code. New posts are written in Markdown in Drafts and added to a new post in VS Code.
